Heads up: Facebook does not provide a public API for creating Marketplace listings from a browser app, so this tool can't auto-post for you. Each card gives you a copy-ready title and description plus a button that opens Facebook's "create listing" page — paste the text in and add your own photos. Price suggestions come from Discogs' current marketplace lowest-price data where available and are estimates, not guarantees.

Network requests go through public CORS proxies (allorigins, codetabs, corsproxy.io) since the Discogs API doesn't send browser CORS headers directly. If a pull fails, wait a few seconds and try again — you may be rate-limited.
